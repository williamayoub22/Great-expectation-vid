# Great Expectations - Unity Creative Task

## Project Description
This repository contains a creative response to Charles Dickens' 'Great Expectations' for an English class assignment. 

Rather than a traditional essay, I have developed a cinematic experience using the Unity Engine to visualize the atmosphere, themes, and settings of the novel in the cemetery scene. 


## How to Run
clone the repo and run the exe. you can also see the video attached to the github repo.
