# Great Expectations - Unity Creative Task

## Project Description
This repository contains a creative task for a highschool English class assignment showing a modled cenematic video of Charles Dickens' 'Great Expectations'.

Rather than a traditional essay, I have developed a cinematic experience using the Unity Engine to visualize the atmosphere, themes, and settings of the cemetery scene in the novel. 


## How to Run
clone the repo and run the exe. you can also see the video attached to the github repo.
